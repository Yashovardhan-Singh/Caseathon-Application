# Caseathon Application

Solution Application for Caseathon 2026 (2nd position)

* Cold Supply Chain Logistics and Tracking (Demo)
* On Site catch recording
* Registering partner vendors
* Margin Calculation, compared to market
* Functional Marketplace (pending transaction API)
* Admin Dashboard for logistics and stastics tracking
* Admin Dashboard also allows catalogue editing
* 90 day target scores for improving pilot traction score
